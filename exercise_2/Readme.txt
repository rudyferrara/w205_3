1) Change to the directory where the storm application resides. For example:

cd /root/EXTwoTweetwordcount

2) Start the application:

sparse run

3) After allowing it to run for a while press ctrl-c to stop it.

4) Type the following commands to test the finalresults.py and histogram.py scripts.

python finalresults.py hello
python finalresults.py
python histogram.py 3,8
